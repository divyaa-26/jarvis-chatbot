🤖 JARVIS - Iron Man Inspired AI Chatbot

An intelligent, persona-driven chatbot inspired by Iron Man’s JARVIS.  
Built with Python and Gradio, this project demonstrates conversational logic, intent detection, and UI design.

---

## ✨ Features

- 🧠 Persona-based chatbot (JARVIS personality)
- 🎯 Intent detection using keyword matching
- 💬 Context-aware conversational replies
- 🎨 Custom UI using Gradio Blocks + CSS
- ⚡ Fast and lightweight (no heavy model required)

---

## 🧠 How It Works

The chatbot follows a **rule-based NLP approach**:

- Detects user intent using keyword matching  
- Maps input → predefined response categories  
- Maintains a consistent JARVIS tone (calm, intelligent, slightly witty)  
- Uses structured functions for clean logic separation  

---

## 🛠️ Tech Stack

- Python  
- Gradio  
- Basic NLP (rule-based system)
